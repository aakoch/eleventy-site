# eleventy-site

Testing [Zach Leatherman's new static site generator](https://github.com/zachleat/eleventy).

Generated with (generally) these commands:
```
npm install eleventy-cli -g
npm install gulp
npm install --save-dev gulp-gh-pages
eleventy
git add .
git commit -m "commit"
gulp deploy
```
